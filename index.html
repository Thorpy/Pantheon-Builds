How builds work

  • Keystones require ALL listed prerequisites, small/medium nodes need only one.
  • Sara and Bandos share 312 of 314 nodes. The root choice affects which nodes are cheaper to reach.
  • Root switches are marked by a new row with a different god.
  • Duplicate rows labelled (same as previous) mean the optimal build hasn’t changed.
  • Temporary prerequisite patches have been applied for nodes that were previously floating (22, 23, 24, 175, 239, 279). These will be removed once the game data is fixed.
  • Devout Vessel = HP bonus from prayer bonus %, not damage.
  • Flurry = next attack fires 1 tick sooner.
  • Crackling Staff halves powered‑staff attack speed.
  • Use the Planner link to see the exact node selection and total cost.
